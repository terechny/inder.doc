

## Типы данных в PHP

В языке PHP существует 10 базовых типов данных, 
которые разделяются на четыре основные категории: скалярные, составные, специальные и псевдотипы. 
PHP — это язык с динамической типизацией, поэтому тип переменной определяется автоматически в момент присвоения ей значения.

Скалярные (простые) типы

* `bool`: Логический тип, принимающий только `true` (истина) или `false` (ложь).
* `int`: Целые числа (например, 42 или -15).
* `float`: Числа с плавающей точкой / дробные числа (например, 3.14).
* `string`: Текстовые строки, заключенные в одинарные или двойные кавычки.

Составные (сложные) типы

* `array`: Массивы, которые могут содержать упорядоченные списки или ассоциативные пары "ключ-значение".
* `object`: Объекты, представляющие собой экземпляры классов в ООП.

Специальные типы

* `null`: Специальное значение, обозначающее переменную без значения.
* `resource`: Ссылка на внешний ресурс (например, открытый файл или подключение к базе данных).

Псевдотипы и служебные типы

* `callable`: Обозначает функции или методы, которые можно вызвать в коде.
* `mixed`: Означает, что переменная может принимать абсолютно любой тип данных. 

##  Переменные в PHP

##### Ссылки (References)

* Суть: Два имени для одного значения.
* Свойства: Позволяют разным переменным ссылаться на одну область памяти.
* Пример: Изменение одной переменной меняет и вторую.
* Синтаксис: Используют символ амперсанда `($a = &$b)`.

##### Глобальные переменные

* Суть: Переменные, созданные вне функций.
* Свойства: Недоступны внутри функций напрямую.
* Доступ: Требуют ключевое слово `global` внутри функции.
* Альтернатива: Доступны через массив `$GLOBALS`.

##### Суперглобальные переменные

* Суть: Встроенные массивы самого PHP.
* Свойства: Доступны автоматически в любом месте кода.
* Область: Видны внутри любых функций и классов без условий.
* Примеры: `$_GET, $_POST, $_SESSION, $_SERVER`

##### Константы

* Суть: Постоянные значения, которые нельзя изменить.
* Свойства: Не имеют знака доллара `$`, глобальны по умолчанию.
* Объявление: Создаются через `define()` или ключевое слово `const`.
* Пример: ` define('API_KEY', '12345'); const VERSION = '1.0.0'`.


## Операторы в PHP

##### Арифметические операторы

* Назначение: Выполняют базовые математические вычисления.
* Основные: `+` (сложение), `-` (вычитание), `*` (умножение), `/` (деление).
* Специфичные: `%` (остаток от деления), `**` (возведение в степень).

##### Операторы сравнения

* Назначение: Сравнивают два значения и возвращают `true` или `false`.
* Равенство: `==` (равно по значению), `===` (тождественно равно: по значению и типу).
* Неравенство: `!=` или `<>` (не равно), `!==` (тождественно не равно).
* Сравнение: `>, <, >=, <=,` а также `<=>` (Spaceship-оператор для сортировки).

##### Логические операторы

* Назначение: Объединяют логические условия.
* Виды: `&& или and (И), || или or (ИЛИ), ! (НЕ), xor (исключающее ИЛИ)`.
* Особенность: `&&` и `||` имеют более высокий приоритет выполнения, чем and и or.

##### Null-операторы (PHP 7+)

* Null-coalescing (`??`): Возвращает левый операнд, 
  если он существует и не `null`, иначе возвращает правый (`$username = $_GET['user'] ?? 'anonymous'`).
* Null-coalescing assignment (`??=`, PHP 7.4+): 
  Присваивает значение правой части, только если переменная слева равна null (`$a ??= 'default'`).

##### Тернарный оператор

* Назначение: Сокращенная форма конструкции `if-else`.
* Синтаксис: Условие `?` `Значение_если_true` : `Значение_если_false`.
* Пример: `$status = ($age >= 18) ? 'adult' : 'minor'`;.

##### Конструкция match (PHP 8.0+)

* Назначение: Современная и строгая альтернатива оператору `switch`.
* Свойства: Возвращает значение, использует строгое сравнение (`===`), не требует `break`.
* Синтаксис: Проверяет выражение и сразу выполняет подходящую короткую ветку кода.

```php

// switch (PHP < 8.0)
switch ($status) {
    case 'active':
        $message = 'Активен';
        break;
    case 'pending':
        $message = 'В ожидании';
        break;
    default:
        $message = 'Неизвестно';
}

// match (PHP 8.0+)
$message = match($status) {
    'active' => 'Активен',
    'pending' => 'В ожидании',
    default => 'Неизвестно',
};

// Несколько значений
$type = match($code) {
    200, 201, 204 => 'success',
    400, 404 => 'client_error',
    500, 502, 503 => 'server_error',
    default => 'unknown',
};

// Условия
$category = match(true) {
    $age < 18 => 'child',
    $age < 65 => 'adult',
    default => 'senior',
};

```

##### Оператор объединения строк

* Назначение: Склеивание (конкатенация) текстовых строк.
* Синтаксис: Точка `.` (например, `'Hello ' . 'World'`).
* Присвоение: `.=` добавляет строку в конец существующей переменной (`$text .= 'добавка'`).

##### Оператор распаковки (Spread, PHP 7.4+)

* Назначение: Распаковывает массивы или коллекции элементов.
* Синтаксис: Три точки `...` перед массивом.
* Применение: Используется для быстрой сборки массивов `[1, 2, ...$more_numbers]` или передачи массива как списка аргументов в функцию.


* Условные конструкции

- `if, elseif, else` : Выполняют код по условию. 
- `if` проверяет истинность. 
- `elseif` дает альтернативу. 
- `else` срабатывает, если всё ложно.

* Сравнение значений

- `switch vs match`: Выбирают код по значению переменной. 
- `switch` делает нестрогое сравнение `(==)`.     
- `match` (с PHP 8.0) строже (===), возвращает значение и компактнее.

* Циклы

- `while, do-while`: Повторяют код, пока верно условию.
- `while` проверяет условие до начала итерации. 
- `do-while` гарантирует хотя бы один запуск кода.
- `for`: Цикл со встроенным счетчиком. 
   Удобен, когда количество повторений известно заранее. Содержит старт, условие и шаг.

- `foreach`: Специальный цикл для перебора массивов и объектов. 
Автоматически извлекает элементы без ручного управления индексами.
Управление циклами и кодом

- `break, continue`: Меняют ход работы цикла. 
- `break` полностью прерывает текущий цикл. 
- `continue` пропускает остаток кода и начинает новый круг. 

Завершение работы: 

- `return, exit, die`: Останавливают выполнение. 
- `return` завершает функцию и возвращает данные. 
- `exit` и `die` (синонимы) мгновенно убивают весь скрипт.

Настройки компилятора:

- `declare(strict_types=1)`: Включает режим строгой типизации. 
   Запрещает PHP автоматически приводить типы данных при вызове функций. Защищает от скрытых ошибок.

## Функции в PHP

##### Объявление и вызов функций

- `function name() {}` создает блок кода. Вызов `name()` запускает его. Имена функций в PHP регистронезависимы.

##### Типизация и аргументы

* `Type Hints`: Указание типов для параметров и возврата. 
  PHP проверяет соответствие типов при вызове функции. Ошибки вызывают `TypeError`.

* `Variadic Functions`: Функции с переменным числом аргументов через оператор `...`. Собирают все лишние переданные аргументы в один массив.

##### Замыкания и короткие функции

* `Анонимные функции (Closures)`: Функции без имени. Передаются как переменные. 
   Используют внешние переменные только через явное указание `use ($var)`.

* `Arrow Functions (PHP 7.4+)`: Короткий синтаксис анонимных функций `fn() => expr`. 
   Автоматически захватывают внешние переменные по значению.

##### Продвинутые концепции

* `Generators` (Генераторы): Функции, использующие `yield` вместо `return`. 
   Возвращают итерируемый объект. Экономят память, выдавая данные по одному.

* Рекурсия: Функция, которая вызывает саму себя внутри своего тела. 
  Требует обязательного условия выхода во избежание бесконечного цикла.

* `Callable` и `First-Class Callable` (PHP 8.1+): 
  `callable` — это тип для любых вызываемых структур. 
  `First-Class` синтаксис `myFunc(...)` создает Closure из любой функции.

## Массивы в PHP

##### Создание и базовая работа

* Создание массивов: Синтаксис `[]` или `array()`. Хранят любые типы данных. Могут быть списковыми (индексы 0, 1, 2) или ассоциативными (ключ => значение).
* Добавление и удаление элементов: `[]=` добавляет в конец, `array_push()` делает то же самое.
  Извлечение: `array_pop()` (с конца), `array_shift()` (с начала).       Удаление: `unset($arr[$key])`.

* Функции высшего порядка

    - `array_map`: Применяет callback-функцию к каждому элементу. Модифицирует значения и возвращает новый массив той же длины.
    - `array_filter`: Фильтрует элементы с помощью callback-функции. Оставляет только те элементы, для которых функция вернула true.
    - `array_reduce`: Последовательно сжимает массив в одно единственное значение (например, сумму), используя callback-функцию и аккумулятор.

* Сортировка и комбинирование: 

  - `sort()` (по значению), 
  - `rsort()` (обратная), 
  - `asort()` (с сохранением ключей), 
  - `ksort()` (по ключам), 
  - `usort()` (пользовательская по callback)

  - `array_merge` сливает массивы. 
  - `array_combine` создает массив, используя один массив для ключей, другой для значений. 
  - `array_diff` находит элементы, которых нет в других массивах.

* Поиск и трансформация:

    - `in_array` проверяет наличие значения. 
    - `array_key_exists` ищет конкретный ключ. 
    - `array_search` ищет значение и возвращает его ключ.

    - `array_column` выбирает одно поле из многомерного массива. 
    - `array_unique` удаляет дубликаты. 
    - `array_flip` меняет ключи и значения местами.

##### Современный синтаксис

* Деструктуризация массивов (PHP 7.1+): 

    - Синтаксис `[$a, $b] = $array`. 
      Быстро раскладывает элементы массива в отдельные переменные. 
      Работает и с ассоциативными ключами.

* `Spread Operator` (PHP 7.4+): 

    Распаковка массива через `...$array`. Позволяет быстро и производительно объединять массивы или передавать элементы в аргументы функций.


## Строки и регулярные выражения

* Работа с подстроками

    - `substr` вырезает часть строки по индексам. 
    - `mb_substr` делает то же самое для многобайтовых кодировок (например, UTF-8). 
    - `str_replace` заменяет одни подстроки на другие.

* Разделение и объединение

    - `explode` разбивает строку в массив по разделителю. 
    - `implode` собирает массив обратно в строку. 
    - `str_split` разбивает строку на массив из отдельных символов заданной длины.

* Поиск и проверка (PHP 8.0+)

    - `strpos` ищет позицию первого вхождения подстроки.
    - `str_contains` проверяет, есть ли подстрока внутри. 
    - `str_starts_with` проверяет, начинается ли строка с указанного фрагмента.

* Функции регулярных выражений

    - `preg_match` ищет совпадения по PCRE-шаблону в строке и возвращает 1 или 0. 
    - `preg_replace` выполняет поиск по регулярному выражению и заменяет найденные совпадения.

##### Основные паттерны регулярных выражений

* Спецсимволы и квантификаторы: 

  - `^` (начало строки), 
  - `$` (конец строки), 
  - `.` (любой символ), 
  - `\d` (цифра),
  - `\w` (буква/цифра), 
  - `*` (0 и более повторений), 
  - `+` (1 и более повторений), 
  - `?` (0 или 1 повторение), 
  - `[a-z]` (диапазон символов).
